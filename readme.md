# hex2tailwind

Converts Hex Colors to Tailwind CSS Color Names

# Installation

```bash
  npm i hex2tailwind
```

## Usage/Examples

```javascript
const Hex2tailwind = require('hex2tailwind');

const hexColor = '#33ccff';
const tailwindClass = Hex2tailwind(hexColor);
console.log(tailwindClass); // Outputs: "sky-500"
```
